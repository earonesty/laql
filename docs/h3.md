# H3

LaQL exposes H3-oriented scalar predicates through expression calls:

```ts
fn("h3_in", col("h3_8"), lit(JSON.stringify(["8829a1d757fffff"])));
fn("h3_within", col("h3_8"), lit("8829a1d757fffff"), lit(1));
```

Sidecar H3 indexes can prune exact-radius `h3_within(..., 0)` calls and `h3_in` calls. Positive-radius `h3_within` pruning is conservative and keeps files planned unless an index can prove safety.
